# Njoro Portfolio (Static)
A fully static, no-backend portfolio using HTML + CSS + Bootstrap + a dash of JS, with:
- Neon dark theme + glow
- Animated skill bars
- Portfolio grid with filters
- **Case Study** pages (SPA-style routing using `#/project/:id`)
- Persistent Navbar + Footer
- Smooth fade/slide page transitions

## Getting Started
Just open `index.html` in a browser or host the folder on Netlify/Vercel/GitHub Pages.

## Customize
- Update your details in `index.html` (hero, contact).
- Edit projects inside `script.js` (`projects` array).
- Tweak colors and glow in `styles.css`.

## Deep Links
You can link directly to a project, e.g. `#/project/1`.
