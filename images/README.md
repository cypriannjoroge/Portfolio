# Portfolio Images CDN

This directory contains portfolio images served via GitHub Pages for faster loading.

## Images
- Portfolio project screenshots
- Logo and profile images
- Optimized for web delivery

## CDN Usage
Images are accessible at: `https://[username].github.io/[repo]/images/[filename]`

## Benefits
- ✅ Free hosting
- ✅ Global CDN
- ✅ HTTPS support
- ✅ Fast loading
- ✅ Version control
